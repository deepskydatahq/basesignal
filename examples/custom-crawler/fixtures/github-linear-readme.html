# Linear Linear is a better way to build software. Streamline issues, projects, and product roadmaps. ## Features - **Issue Tracking**: Create, assign, and track issues with keyboard-first navigation and sub-second performance. - **Projects**: Organize work into projects with timelines, milestones, and progress tracking. - **Cycles**: Plan sprints with automatic carry-over and velocity tracking. - **Roadmaps**: Visualize product direction across teams and time horizons. - **Views**: Custom filtered views for any workflow (triage, backlog, my issues). - **Integrations**: GitHub, GitLab, Slack, Figma, Sentry, and 50+ more. ## Getting Started ### Sign Up 1. Go to [linear.app](https://linear.app) 2. Create a workspace 3. Invite your team ### API Access Linear provides a GraphQL API for programmatic access: ```bash curl -X POST https://api.linear.app/graphql \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{"query": "{ viewer { id name email } }"}' ``` See the [API documentation](https://developers.linear.app) for full reference. ## Why Linear? - **Speed**: Sub-50ms interactions. No loading spinners. - **Keyboard-first**: Every action has a shortcut. Power users never touch the mouse. - **Opinionated workflows**: Cycles, triage, and backlog management built in. - **Beautiful design**: A tool teams actually want to use. ## Resources - [Documentation](https://linear.app/docs) - [API Reference](https://developers.linear.app) - [Changelog](https://linear.app/changelog) - [Blog](https://linear.app/blog) ## License Copyright (c) Linear Orbit, Inc.
